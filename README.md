# DevBoard

A single-user developer dashboard: projects, tasks, deadlines, and live GitHub
repository data (commits, issues, pull requests) in one workspace.

**Stack:** TypeScript, Express 5, PostgreSQL (Drizzle ORM), React + Tailwind (coming), GitHub REST API.

## Roadmap
- [x] Phase 1 — Backend scaffold: Express + TS, env validation, Drizzle client, health endpoint, Postgres via Compose
- [ ] Phase 2 — Projects & tasks CRUD
- [ ] Phase 3 — GitHub integration (commits, issues, PRs) with caching
- [ ] Phase 4 — React + Tailwind scaffold
- [ ] Phase 5 — Dashboard UI: reusable components, search/filter
- [ ] Phase 6 — GitHub panels + responsive pass
- [ ] Phase 7 — Tests (Vitest + Supertest)
